# page

Используется для задания стилей всей страницы. Применяется к тегу `body`.

## Примеры

```html
<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="utf-8" />
    <meta
      name="viewport"
      content="width=device-width, initial-scale=1, shrink-to-fit=no"
    />

    <title>Title</title>

    <link rel="stylesheet" href="main.css" />
  </head>
  <body class="page">
    <script src="main.js"></script>
  </body>
</html>
```
